Title of the project:
agriculture intelligente

Description:
 afficher les valeur de : temperatur , humidite , humidite sol
envoyer les donnees a l'interface web avec l'mqtt
controller la pomp et le port avec l'interface web

Objectif:
ce projet facilité de l'agriculture et permet de le contrôler(l'ouverture de la porte et la pompe) 
à travers d'une page web qui permet d'afficher la température, lhumedité et lhumedité sol
List of devices:
Esp32
Capteur DHT
Capteur humidite sol
Servo Moteur
Pomp Elecrique
carte raspberry pi


Installation and prepartion instrcutions: 
installer des bublioteques : wifi/DHT/pubsubclient/esp32servo
installer mosquitto on raspberry pi
